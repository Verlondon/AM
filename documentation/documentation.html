A) Installation top

There is no installation needed for this Theme, as it's meant to be a model for experienced users to customize the theme and set it up for their website (there is a WordPress version of this theme currently in development).

The theme will work as is. If you upload the files and manually call the "index.html" the theme will be functional with content that demonstrates its capabilities.

C) CSS top

This template is built with CSS2.1/CSS3. The main Stylesheet is a file called "style.css."" This file contains the majority of the front-end design code.

There is a boilerplate file called "main.css" which includes other foundational settings. You can view details about this boilerplate here.

Lastly, there is another general correction file named "normalize.css." This file also contributes to the foundation of the theme and you can view more details about this file here

The Stylesheet sections of the CSS are clearly marked by these comments within the file:

  • Global Styles
  • Menu Styles
  • Header Styles
  • Body Structure and Page Headers
  • Home
  • About Us
  • Portfolio
  • Articles
  • Articles Light Box
  • Key Points
  • Footer
  • Media Queries
  • Swipe Slideshow Styles

If you would like to edit this file, find the appropriate comment in the CSS file then scroll down until you find the style that needs to be edited.

D) Javascript top

Listed below are the different key files of javascrpit within this theme along with their filename and a brief description:

  • Main - "fluid/js/main.js" - This file contains the core functionality for the theme. See document comments for details.
  • jQuery - "//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js" - Using JQuery version 1.10.2.
  • Mousewheel - "fluid/js/vendor/jquery.mousewheel-3.0.6.pack.js" - This file contains the main code for the scroll wheels of the theme.
  • Fancybox - "fluid/js/vendor/jquery.fancybox.js?v=2.1.5" - This is the file containing the main code for the light-boxes of the theme.
  • Fancybox-Thumbs - "fluid/js/vendor/jquery.fancybox-thumbs.css?v=1.0.7" - Related to the main fancybox file, this contains the settings for light-box thumbnails.
  • Fancybox-Media - "fluid/js/vendor/jquery.fancybox-media.js?v=1.0.6" -
  • Plugins - "fluid/js/plugins.js" - This space leaves room for any user to add jquery plugins to the code.
  • Slides - "fluid/js/jquery.slides.min.js" - This file contains the main code for various sliders in the theme.
  • html5Shiv - "fluid/js/html5shiv.js" - This Shiv is a workaround that makes sure older browsers correctly recognize HTML5 code.
  • jCarousel Lite - "jcarousellite_1.0.1.min.js" - This plugin contributes to the carousels of the template and you can find more info about it here.

E) Images top

All of the images and icons featured in the theme are located in the folder named "images."

If you navigate into the subfolder named "icons" you will find more subfolders with icons and images from each section of the theme.

F) PSD File top

Provided with this theme is a PSD file containing dozens of unique elements. The file is layered, organized well, and ready for your use.

The file is located in the folder titled "psd" and the file name is "fluid-psd.psd"

G) 404.html top

This file contains the code for the theme 404 page.

You can edit the file at this location: "fluid/404.html"
